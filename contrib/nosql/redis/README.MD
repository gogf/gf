# redis
GoFrame `gredis.Adapter` implements using `go-redis`.

# Installation
```
go get -u github.com/gogf/gf/contrib/nosql/redis/v3
```

Commonly imported at top of `main.go`:
```go
package main

import (
	_ "github.com/gogf/gf/contrib/nosql/redis/v3"

	// Other imported packages.
)

func main() {
	// Main logics.
}
```
